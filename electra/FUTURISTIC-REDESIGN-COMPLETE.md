## Design System

### Color Palette:
- **Cyber Blue** (#2979FF) - Primary tech color
- **Electric Green** (#00FF94) - Energy/power color
- **Cyan** (#00D9FF) - Accent color
- **Alert Orange** (#FF9800) - Threat detection

### Animation Patterns:
- **Rotating Icons** - 360° rotation (4-20s duration)
- **Pulsing Glows** - Box shadow animations (2s cycles)
- **Expanding Rings** - Scale from 1 to 1.2-1.3
- **Scanning Lines** - Linear movement across containers
- **Particle Effects** - Floating with opacity changes
- **Shimmer Effects** - Gradient movement across elements

### Typography:
- **Headers** - 3xl (30px) with gradients
- **Subtitles** - xs (12px) uppercase with tracking
- **Monospace** - For technical labels
- **Gradient Text** - Cyber → Electric → Cyan

### Container Design:
- **Rounded Corners** - rounded-3xl (24px)
- **Borders** - 2px solid with 30-40% opacity
- **Background** - Gradient from dark-card to dark-base
- **Backdrop** - Grid/circuit patterns at 5% opacity
- **Corner Accents** - Blur-3xl glowing orbs

### Interactive Elements:
- **Hover Scale** - 1.05-1.1x
- **Hover Lift** - y: -2px
- **Tap Scale** - 0.95-0.98x
- **Smooth Transitions** - 0.3-0.5s duration
